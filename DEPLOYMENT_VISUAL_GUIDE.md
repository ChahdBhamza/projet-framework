## Common Issues & Quick Fixes

| Issue | Solution |
|-------|----------|
| **Frontend shows API error** | Check `NEXT_PUBLIC_API_URL` in Vercel env vars |
| **502 Bad Gateway** | Verify Koyeb backend is running: `/api/health` |
| **MongoDB connection timeout** | Check IP whitelist in MongoDB Atlas |
| **CORS errors** | Verify Vercel domain in Koyeb CORS settings |
| **Frontend calls localhost** | Update `.env.local` with correct backend URL |
| **Koyeb deployment fails** | Check `npm run start` works locally |
